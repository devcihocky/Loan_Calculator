Converting Celsius to Fahrenheit

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads a temperature value in Celsius as input and prints the equivalent temperature in Fahrenheit as output. To convert Celsius to Fahrenheit, multiply the Celsius value by 9, divide the result by 5, and then add 32.
Sample Input:
0

Sample Output:
32.0


Sample Input:
100

Sample Output:
212.0



Memory limit: 256 MB
Time limit: 15 seconds
