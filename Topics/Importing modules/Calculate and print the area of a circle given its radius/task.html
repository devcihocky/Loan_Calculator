Calculate and print the area of a circle given its radius

Write a program that uses Python's built-in math module. Your program should read a single float value from input, which represents the radius of a circle. Using the math module's 'pi' constant and the provided radius, your program should calculate and print the area of the circle. The area of a circle is calculated using the formula pi*r^2.

Round the result to show 2 digits after the decimal point.


Sample Input:
5

Sample Output:
78.54


Sample Input:
7.1

Sample Output:
158.37



Memory limit: 256 MB
Time limit: 15 seconds
